# Ringkasan Proyek

## 1. ClassicML
**ClassicML** adalah library Machine Learning ringan yang ditulis sepenuhnya dalam **C#** dari awal. Library ini tidak bergantung pada library matematika eksternal, menjadikannya sempurna untuk tujuan edukasi. Beberapa algoritma yang diimplementasikan meliputi:
- **Supervised Learning**: Linear Regression, Logistic Regression, k-NN, Naive Bayes, SVM, Decision Tree, Random Forest.
- **Unsupervised Learning**: k-Means, PCA, Hierarchical Clustering.
- **Neural Approaches**: Perceptron, Multilayer Perceptron.
- **Reinforcement Learning**: Multi-Armed Bandit, Q-Learning.

## 2. D3Net
D3Net adalah library visualisasi data lintas platform yang terinspirasi dari D3.js, dibangun menggunakan **Avalonia UI Framework**. Library ini menyediakan 10 jenis chart dengan animasi halus dan grafis menarik. Fitur utamanya meliputi:
- **Cross-Platform**: Berjalan di Windows, Linux, dan macOS.
- **10 Chart Types**: Berbagai jenis visualisasi.
- **Smooth Animations**: Animasi halus dengan API Animasi Avalonia.

## 3. MicroUI
MicroUI adalah framework UI ringan dan lintas platform yang dibangun menggunakan C# untuk ekosistem .NET. Proyek ini dirancang untuk mensimulasikan pengembangan UI pada sistem tertanam. Fitur utamanya meliputi:
- **Ringan & Sederhana**: Struktur kontrol yang mudah dipahami.
- **Dukungan Tata Letak XML**: Mendesain antarmuka menggunakan file XML.
- **Simulator Cross-Platform**: Berjalan di mana saja .NET dan Avalonia didukung.

## 4. NetScheduler
NetScheduler adalah proyek perpustakaan scheduler berbasis .NET Console Application yang dirancang untuk mendemonstrasikan kemampuan penjadwalan tugas tingkat lanjut. Fitur utamanya meliputi:
- **Job Scheduling & Execution**: Mendukung interval sederhana dan ekspresi Cron.
- **Persistence**: Mendukung stateful job.
- **Concurrency Control**: Mencegah eksekusi tumpang tindih.
